The **Rock Paper Scissor Game** is a Python console game where the user competes against the computer for a chosen number of rounds. The computer randomly picks Rock, Paper, or Scissor each round.

This project demonstrates :
- Random selection  
- Score tracking (User, Computer, Draws)  
- Looping through multiple rounds  
- Clean input validation  
- Replay functionality

## ğŸ® How the Game Works
1. User selects the number of rounds to play  
2. For each round :
   - User picks Rock, Paper, or Scissor  
   - Computer randomly selects one  
   - Winner is determined based on standard rules  
3. After all rounds :
   - Final scoreboard is displayed  
4. User can choose to play again
